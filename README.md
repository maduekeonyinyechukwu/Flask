# Flask Task Manager 📝

A simple task management web application built with Flask and SQLAlchemy. This app demonstrates basic **CRUD** (Create, Read, Update, Delete) functionality using a SQLite database.

## Features

- ✅ Add new tasks
- 📋 View all tasks
- ✏️ Edit/update tasks
- 🗑️ Delete tasks

## Technologies Used

- Python 3.12
- Flask
- Flask SQLAlchemy
- SQLite
- HTML (Jinja2 templates)
